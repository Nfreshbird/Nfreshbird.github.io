linux命令-查看文件夹信息

查询文件夹信息

  • 查看当前目录下所有文件的详细信息

  • 查看当前目录下文件的个数(不包含目录)

  • 查看当前目录下文件的个数(包含子目录)

  • 查看目录下文件夹(目录)的个数(包含子目录)

命令解析

  • ls -l
    长列表输出该目录下文件信息，这里的文件包含目录、链接、设备文件
    ，每一行对应一个文件或目录，ls -lR 列出所有文件 包含目录

  • grep “^-“
    过滤，只保留一般文件,只保留目录，使用”^d”

  • wc -l
    统计输出信息的行数，一行对应一个文件，所以就是文件的个数

如何使用lambda(python)

enjoy learning , enjoy life.

lambda匿名函数并不是python所独有的特点，很多其他语言也有这个特性，匿名函数有之所以被设计出来，我想最重要的原因就是编写简单的函数时不必再花时间取名字。

lambda函数最常应用的地方就是作为一个函数对象，被其他函数调用，对列表中的大量元素集中处理。

lambda函数语法 []代表可有可无

lambda [param1 [,param2,param3]]: expression

lambda函数有以下几个特性

  • 如果lambda表达式返回的函数对象没有对象接收，会被立即丢弃
  • lambda 在：后面 只能有一个表达式，可以使用if else，需要符合特定语法
  • lambda 不能被别的程序调用

普通用法

demo1 单个参数

demo2 多个参数

demo3 类似于函数，也可以使用默认参数

进一步使用

demo4 lambda表达式可以作为函数对象被存放在列表中

demo5 带if else条件的lambda表达式

lambda text: int(text) if text.isdigit() else text.lower()
这段代码，将传入的text进行判断，如果是数字转化为数字，这里可以看上面的结果是23，而不是”23”,如果不是数字，则全部转化为小写。

这里的lambad使用if else 条件时必须满足以下语法
lambda x : 语句1 if 条件1 [else 语句2 if 条件2][else 语句x if条件x] else 最后的语句

实际上是这样的语法表示
lambda x : 语句1 if 条件1 else (语句2 if 条件2 else 语句3)

总之 if 和 else 的数量必须相等
